The Repository
I’m currently working a series of contracts for AudioVerse building a WordPress plugin which reproduces the functionality of their current website. Goals are to make the website easier to update for their non-technical staff and to eventually take advantage of PWA browser APIs to reduce their reliance on maintaining separate mobile apps. Locally I use Docker as my development environment.

I'm building a WordPress plugin for AudioVerse. I do all the planning and then I send them a contract and then I do the work. I built a router, I built a system for them to use to place content on different media pages. I built short codes. I architected the whole thing. It's written using test-driven development.
